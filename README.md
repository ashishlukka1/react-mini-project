## Overview

A clone of the Swiggy food delivery platform built for learning and demonstration purposes.

## Features

- User authentication
- Restaurant listing
- Menu browsing
- Cart management
- Order placement

## Installation

```bash
git clone https://github.com/ashishlukka1/react-mini-project.git
cd swiggy-clone
npm install
```

## Usage

```bash
npm start
```

Visit `http://localhost:3000` in your browser.
